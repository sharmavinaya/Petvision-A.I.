# Petvision-A.I.
PetVision AI is a real-time object detection system using YOLOv8 and Flask to identify cats, dogs, and humans through webcam input with dynamic bounding boxes and motion-based prediction.

# PetVision AI 🧠🐶👤

Real-time AI detection system .

## Features
- Detects Dogs, Cats, Humans
- Real-time webcam detection
- Motion-based detection

## Tech Stack
- Flask
- YOLOv8
